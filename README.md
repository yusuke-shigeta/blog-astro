# node.js をインストール

- バージョン: `v18.14.1` 以降
- [nodejs 公式からダウンロード](https://nodejs.org/ja)
  ```
  This package will install:
    •Node.js v22.13.1 to /usr/local/bin/node
    •npm v10.9.2 to /usr/local/bin/npm
  ```
- インストールしたバージョン: `v22.13.1`
