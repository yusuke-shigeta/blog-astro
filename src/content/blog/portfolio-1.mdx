## docker Desktop インストール

[参考記事](https://nosubject.io/windows11-how-to-setup-wsl2-ubuntu-and-docker-desktop/)

- wsl をアップデート
  - wsl --update
- Ubuntu を追加
  - `wsl --install -d Ubuntu`
- PC 再起動
- Docker 起動
  - 変だったらアンインストールして再度インストール

## 環境構築

### WordPress 環境構築用のディレクトリを pull

- work-colead ディレクトリを作成
  - desktop/workspace/work-colead/
- work-colead をリポジトリ化
  - `git init`
- WordPress 環境構築用のディレクトリを pull
  - `git pull https://github.com/yusuke-shigeta/setup-wordpress.git`

### docker コマンドでコンテナ作成

- docker desktop を起動
- docker コンテナを構築
  - `docker-compose up -d --build`

### 実際に動くか検証

- 開発環境にアクセス
  - `http://localhost:8001`
  - `http://localhost:8081`
